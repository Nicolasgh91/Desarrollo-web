Guía de Favicon

Los favicons son pequeños íconos que aparecen en la pestaña del navegador y en la lista de marcadores. Aquí se presentan los formatos más populares y cómo implementarlos en tu sitio web.

Formatos de Favicon

Formato Extensión Tamaño Recomendado Uso
ICO .ico 16x16 px, 32x32 px, 48x48 px Compatible con todos los navegadores.
PNG .png 32x32 px, 96x96 px, 192x192 px Soporta transparencia, recomendado para Android y Web App.
SVG .svg Variable Usado para iconos escalables en aplicaciones modernas.

Implementación en HTML

Para incluir los favicons en tu sitio web, agrega las siguientes líneas en la sección <head> de tu documento HTML:

Ejemplo para Favicon ICO

<link rel="icon" href="favicon.ico" type="image/x-icon">

Ejemplo para Favicon PNG

<link rel="icon" href="favicon.png" type="image/png" sizes="32x32">

Ejemplo para Favicon SVG

<link rel="icon" href="favicon.svg" type="image/svg+xml">

Consejos

  • Usa un formato ICO como estándar para garantizar la compatibilidad en todos los navegadores.
  • Proporciona múltiples tamaños de favicon PNG para adaptarse a diferentes dispositivos y resoluciones.
  • Utiliza SVG para iconos escalables que se vean bien en cualquier tamaño.
